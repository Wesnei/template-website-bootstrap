# Template Portfólio Bootstrap 5

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como material didático para alunos do Ensino Médio que estão iniciando seus estudos em desenvolvimento web utilizando **HTML, CSS, Bootstrap 5 e AOS (Animate On Scroll)**.

O objetivo é fornecer uma página pronta contendo diversos componentes modernos para que os alunos possam analisar o código, modificar os elementos e aprender na prática como construir interfaces responsivas, interativas e visualmente atrativas.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* AOS (Animate On Scroll)

---

## ✨ Componentes e Recursos Utilizados

Neste template você encontrará exemplos dos seguintes componentes e recursos:

### Bootstrap 5

* Navbar (Menu de Navegação)
* Carousel (Carrossel de Imagens)
* Sistema de Grid (Rows e Columns)
* Cards
* Formulários
* Accordion (Perguntas Frequentes)
* Botões
* Containers
* Classes Utilitárias
* Responsividade

### AOS (Animate On Scroll)

Também foram adicionadas animações utilizando a biblioteca AOS, permitindo que elementos apareçam com efeitos durante a rolagem da página, como:

* Fade Up
* Fade Left
* Fade Right
* Flip Up
* Zoom In
* Zoom In Down
* Zoom In Left

Esses efeitos ajudam os alunos a compreender como tornar interfaces mais dinâmicas e modernas.

Além disso, há exemplos de CSS personalizado para complementar a estilização do Bootstrap.

## 🎯 Objetivos de Aprendizagem

Ao estudar este template, você poderá:

* Entender a estrutura básica de uma página web.
* Aprender a utilizar o Bootstrap através de CDN.
* Integrar bibliotecas externas como o AOS.
* Criar layouts responsivos.
* Trabalhar com componentes prontos do framework.
* Adicionar animações aos elementos da página.
* Personalizar estilos utilizando CSS.
* Desenvolver um portfólio próprio.

---

## 🛠️ Como Utilizar

1. Faça o download ou clone o projeto.
2. Abra a pasta no Visual Studio Code.
3. Execute o arquivo `index.html` no navegador.
4. Analise os componentes existentes.
5. Observe os efeitos de animação durante a rolagem.
6. Modifique textos, imagens e cores.
7. Crie novas seções e componentes.
8. Transforme o template em seu próprio portfólio.
